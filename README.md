# Bus Passenger Tracking System - Frontend

React-based frontend for the Bus Passenger Tracking System.

## Prerequisites

- Node.js (v14 or higher)
- npm or yarn

## Installation

```bash
npm install
```

## Configuration

1. Copy `.env.example` to `.env.development`
2. Update the API URL:
   - `VITE_API_URL`: Backend API URL

## Running the App

```bash
# Development mode
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Features

- Real-time passenger tracking
- Trip management
- Season ticket validation
- Face recognition integration
- Fare calculation

## Tech Stack

- React
- Vite
- Axios
- React Router
